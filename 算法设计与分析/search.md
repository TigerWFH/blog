# 查找算法相关知识
## 查找的概念
```
根据给定的某个值，在查找表中确定一个其关键字等于给定值的数据元素（或记录）。
```
## 查找算法分类
* `静态查找和动态查找`
```
静态或动态是针对查找表而言的。
动态表指查找表中有插入和删除操作的表。
```
* `无序查找和有序查找`
```
无需查找：被查找数列有序无序均可以。
有序查找：被查找数列必须为有序数列。
```
