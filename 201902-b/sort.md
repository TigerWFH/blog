# 排序算法相关知识
## 排序的概念

## 内排序算法
```
是指所有的数据已经加载到内存，在内存中进行排序的算法。排序过程中不需要对磁盘进行读写，同时内排序一般也假定所有用到的辅助空间也可以直接存在于内存中。
```

## 外排序算法
```
与内排序算法对应，另一类排序算法称作外排序，即内存中无法保存全部数据，需要进行磁盘读写，每次读入部分数据到内存进行排序。
```
